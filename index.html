Quick Start

1

Download AMCP

Get the latest release or clone from GitHub

git clone https://github.com/agentmeshcommunicationprotocol/amcpcore.github.io.git
2

Build & Run

Compile and start the interactive CLI

mvn clean package
java -jar cli/target/amcp-cli-1.5.0.jar
3

Activate Agents

Start using multi-agent capabilities

amcp> activate weather
amcp> weather London
amcp> activate orchestrator
