üíª Code Examples

Agent Creation & Activation

// Create and activate a Weather Agent
WeatherAgent weatherAgent = new WeatherAgent("weather-001");
agentContext.activateAgent(weatherAgent);

// Subscribe to weather topics
weatherAgent.subscribe("weather.request.*");
weatherAgent.subscribe("system.control.*");

Simple agent creation with topic subscriptions for event-driven communication.

Event Publishing

// Publish weather update event
Event weatherEvent = Event.builder()
    .topic("weather.update.london")
    .payload(weatherData)
    .sender(agentId)
    .build();

eventMesh.publish(weatherEvent);

Asynchronous event publishing to the mesh with structured event format.

Agent Migration

// Migrate agent to edge context
MigrationRequest request = MigrationRequest.builder()
    .agentId("analyzer-001")
    .targetContext("edge-device-7")
    .preserveState(true)
    .build();

agentContext.migrate(request);

Strong mobility with state preservation for dynamic topology management.
